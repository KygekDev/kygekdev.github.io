Creations (KygekTeam)

KygekRanksUI

A PocketMine-MP plugin to showcase ranks currently on sale in a PocketMine-MP server by KygekTeam. Being one of the most popular PocketMine-MP plugin by KygekTeam, KygekRanksUI has many features to cover the need of PocketMine-MP servers to show their ranks currently on sale.

  • More Info on Poggit

KygekTagsShop

A PocketMine-MP plugin to allow players to use tags by KygekTeam. Currently the most advanced plugin by KygekTeam, with API for other plugins depending on it, other languages support and new features to be added in the future.

  • More Info on Poggit

KygekJoinUI

A PocketMine-MP plugin to show an UI for information and guides of the server when players joins a PocketMine-MP server orginally by KygekDev. It is the first plugin by KygekDev, published on Poggit in July 19th, 2020, which is now currently owned and developed by KygekTeam. A full rewrite is planned in the future for better development.

  • More Info on Poggit

KygekWhitelistKick

A PocketMine-MP plugin to kick not whitelisted players when server whitelist turned on by KygekTeam. One of most useful plugins by KygekTeam, as PocketMine-MP doesn't kick not whitelisted players when server whitelist is enabled.

  • More Info on Poggit

KygekEatHeal

A PocketMine-MP plugin to eat (feed) and heal a player instantly by KygekTeam. Supports paid usage using EconomyAPI, customizeable heal and feed amount, and more interesting features.

  • More Info on Poggit

KygekPingTPS

A PocketMine-MP plugin to see server TPS and a player's ping by KygekTeam. Works both in console and players and implements instant configuration settings application without having to restart server.

  • More Info on Poggit
